## 1. 개요

### 1.1 연구 배경 및 동기

UX 연구에서 페르소나(persona)는 사용자 집단을 대표하는 가상 인물로 활용된다. 최근 NVIDIA의 Nemotron-Personas-Korea처럼 LLM이 대규모 합성 페르소나를 생성하는 사례가 늘고 있으나, 이렇게 생성된 데이터가 인구통계 고정관념을 재생산하는지, 또는 실제 행동·가치관 다양성을 포착하는지는 검증된 바 없다.

본 연구는 LLM 생성 합성 페르소나에서 **인구통계 레이블 없이** 행동 텍스트와 가치관 텍스트만으로 의미 있는 사이코그래픽(psychographic) 군집을 추출할 수 있는지 실증적으로 검토한다. 나아가 생성된 군집이 인구통계와 어느 정도 연관되는지, LLM 생성 과정에서 특정 언어 패턴이 과잉 생성되는지를 분석한다.

### 1.2 연구 질문

1. **RQ1**: 행동 텍스트만으로 인구통계와 독립적인 사이코그래픽 군집 추출이 가능한가?
2. **RQ2**: 가치관 텍스트(Layer A)와 행동 텍스트(Layer B)는 독립적인 레이어를 형성하는가?
3. **RQ3**: LLM이 생성한 한국어 페르소나에서 사회성(sociality) 키워드가 과잉 표현되는가?

### 3.3 UMAP 파이프라인 효과

K-Means를 직접 768D 공간에서 적용하면 차원의 저주(curse of dimensionality)로 Silhouette이 낮게 나타난다. UMAP으로 50D로 축소한 뒤 군집화하면 분리도가 대폭 개선된다.

| 단계 | Silhouette |
|------|-----------|
| 원본 768D | 0.0534 |
| UMAP 50D | **0.6018** |
| **개선율** | **+1,027%** |

→ 768D → UMAP 50D 파이프라인의 필요성 정량 입증.

> - 사회성 키워드(함께·친구·가족 등) 평균 4.37회/행 vs 자율성(혼자·스스로 등) 0.27회/행 → **16.1배 차이**

### 4.4 반전 분석 A: 가치관 군집 × 인구통계

#### Cramér's V 효과 크기

| 인구통계 변수 | χ² | Cramér's V | 효과 크기 | 논문 함의 |
|-------------|-----|-----------|----------|----------|
| 연령대 | 3,620.54 | **0.294** | 작음 | 연령으로 가치관 군집 예측 어려움 |
| 성별 | 1,040.36 | **0.315** | 중간 | 일부 연관 존재, 한계점 명시 |
| **지역** | **11,346.66** | **0.520** | **큰** | **주목 발견 — 지역 문화 차이 또는 생성 편향** |
| 학력 | 3,274.33 | **0.279** | 작음 | 학력 기반 AI 분류 근거 없음 |

모든 χ² 검정: p < 0.0001 (대규모 표본 n=10,500에서 미미한 연관도도 유의하게 검출됨).

**인구통계 예측력 테스트**: 로지스틱 회귀 5-fold 정확도 **39.8% ± 10.6%** vs 무작위 기준 **20.0%** (+19.8%p). 통계적으로 유의하나 완전한 설명력은 없음.

> **지역 효과(V=0.520)**: 이 연구의 예상치 못한 주요 발견.
> - **(a) 생성 편향**: Nemotron 생성 시 지역 코드별 다른 프롬프트 적용 가능성
> - **(b) 실제 지역 차이**: 한국 사회의 지역별 가치관 차이 반영 가능성
> → NB04 시뮬레이터에서 지역별 응답 패턴 비교로 검증 예정.

## 5. 통합 논의

### 5.1 NB02 vs NB03 비교

| 항목 | NB02 (행동 군집) | NB03 (가치관 군집) |
|------|-----------------|------------------|
| 입력 텍스트 | Layer B (행동 5컬럼) | Layer A (가치관 3컬럼) |
| 표본 크기 | 21,000행 | 10,500행 |
| Silhouette | **0.6018** | 0.3015 |
| Davies-Bouldin | **0.5551** | 1.1805 |
| 최적 K | 4 | 5 |
| 군집 안정성 (ARI) | **1.0000** | — |
| 군집 분포 | 불균등 (65.1%~2.5%) | 균등 (28.7%~15.3%) |

행동 텍스트는 가치관 텍스트보다 더 선명하게 군집화된다(Silhouette 0.6018 vs 0.3015). 이는 행동 기술이 더 구체적이고 어휘가 다양한 데 기인할 것으로 해석된다.

### 5.2 핵심 주장 3개 실증 요약

**Claim 1 (수정): 인구통계의 예측력은 제한적이나 완전한 독립은 아님**
- Cramér's V: 연령 0.294(작음)·학력 0.279(작음)·성별 0.315(중간)·지역 0.520(큰)
- 로지스틱 회귀 39.8% vs 기준선 20% → 예측력 존재하나 결정적이지 않음
- **지역 효과(V=0.520)**: 생성 편향 또는 실제 지역 차이, NB04 이후 해석 확정 예정

**Claim 2 (견고): 가치관과 행동은 독립적 레이어**
- ARI=0.0287, NMI=0.0378 (랜덤 수준에 근접)
- 교차표 대각선 18.3% < 무작위 기대값 20.0%
- Inter-layer 유사도 0.417 < Intra-A 0.608 / Intra-B 0.565

**Claim 3 (조건부 견고): 사회성 키워드 과잉 생성**
- 16.1배 차이(raw), 키워드 수 정규화 후 10.7배
- 한계: 명사 추출 기반으로 자율성 동사·부사 표현 미포착
- 실제 한국 사회의 집단주의 문화 반영 가능성 배제 불가

### 5.3 LLM 생성 편향의 함의

Nemotron-Personas-Korea는 행동 텍스트에서 사회성 키워드를 자율성 대비 10.7~16.1배 과잉 생성한다. 이 데이터를 다시 LLM 프롬프트로 사용하면 사회성 편향이 증폭될 위험이 있다. 또한 지역 변수의 대 효과(V=0.520)는 지역 코드별 생성 프롬프트 차이가 가치관 분포에 영향을 줬을 가능성을 시사한다. Phase 2(NB04)에서 이 편향 전파 경로를 시뮬레이션할 예정이다.

## 6. 한계점

**지역 변수 효과(V=0.520) 미해결**
Cramér's V=0.520은 중-대 효과로, 이 연구에서 가장 예상치 못한 발견이다. 두 가지 해석 — Nemotron 생성 시 지역별 프롬프트 차이로 인한 생성 편향, 또는 실제 한국 지역별 가치관 차이 — 이 현재 데이터만으로는 구분되지 않는다. NB04에서 지역별 페르소나를 동일 프롬프트에 입력해 응답 분포를 비교함으로써 검증 예정이다.

**동일 임베딩 모델로 인한 레이어 간 표현 공유 가능성**
Layer A와 Layer B 모두 ko-sroberta 모델로 임베딩했다. 동일 모델의 표현 공간을 공유하므로 Inter-layer 유사도(0.417)가 실제보다 높게 측정될 수 있다. 레이어별 전용 파인튜닝 모델 사용이 이상적이나, 본 연구에서는 단일 모델을 사용했다.

**합성 데이터 기반 — 실제 한국인 분포와의 차이 미검증**
Nemotron-Personas-Korea는 실제 설문이나 관찰 데이터가 아닌 LLM 생성 합성 데이터다. 인구통계 분포는 공식 통계에 맞춰 설계됐으나, 행동·가치관 텍스트가 실제 한국인의 다양성을 얼마나 반영하는지는 검증되지 않았다.

**자율성 키워드 명사 중심 설계로 인한 과소계산**
자율성 키워드(`혼자`, `스스로`, `독립`, `개인`)는 명사 형태 중심으로 설계됐다. 한국어에서 자율성 표현은 `혼자 한다`, `자율적으로 결정한다` 같은 동사·부사 형태가 많아, 명사 추출 기반 계산이 자율성 출현 빈도를 구조적으로 과소계산할 수 있다. NB04에서 원문 기반 재검증 예정이다.

## 8. 편향 전파 시뮬레이터 및 GAP 분석 (심층 분석)

### 8.1 NB01: 인구통계 기반 NLP 탐색 및 합성 데이터의 특성

합성 페르소나 100만 명의 텍스트 데이터를 연령, 성별, 지역별로 교차 분석하여 기초적인 키워드 지도를 구축했습니다.

* **주요 발견**:
* **연령별 동질성**: 70대 이상 고령층에서는 성별에 따른 차이보다 '연령' 자체의 특성(건강, 전통, 소박한 일상)이 텍스트 지표를 압도했습니다. 반면, 30대에서는 남녀 간 키워드 간극이 가장 작게 나타나 활동 양상의 유사성이 확인되었습니다.
* **합성 데이터의 한계**: 실제 한국인의 구어체보다는 정제된 문어체 패턴이 반복되는 '텍스트 균질성'이 관찰되었으며, 형태소 분석 시 신조어 처리의 노이즈가 일부 발견되었습니다.



> ![3-way WordCloud Analysis](image/nb01/cross_age_sex_keywords.png)
> *그림 1. 연령 × 성별 × 지역 키워드 분석 - 연령대가 높아질수록 성별 변수보다 연령 변수가 텍스트 분포에 미치는 영향이 커짐을 시각화함.*

### 8.2 NB02: 행동 기반 사이코그래픽 군집화와 사회성 편향

인구통계 레이블을 제외하고 "무엇을 하는가(What)"에 해당하는 행동 텍스트만으로 사용자를 분류했습니다.

* **방법론적 성과**: 768D의 고차원 임베딩을 UMAP 50D로 축소하여 K-Means를 적용한 결과, Silhouette 스코어가 기존 대비 1027% 향상(0.0534 → 0.6018)되어 군집의 응집도를 확보했습니다.
* **사회성 편향(Sociality Bias)**: LLM 생성 데이터에서 '함께', '이웃', '가족' 등 사회성 키워드가 '혼자', '독립' 등 자율성 키워드보다 **평균 16.1배 과잉 생성**되는 경향을 정량적으로 입증했습니다. 이는 LLM이 한국인을 묘사할 때 '공동체 중심적'이라는 고정관념을 강하게 투영하고 있음을 시사합니다.

> ![Image: UMAP Clustering & Bias Bar Chart](그림 2. (좌) 행동 기반 UMAP 군집 시각화 / (우) 사회성 vs 자율성 키워드 출현 빈도 비교 - LLM의 사회성 키워드 편향이 뚜렷하게 나타남)

### 8.3 NB03: 가치관(Why)과 행동(What)의 독립성 검증

사용자의 내면적 지향점(Layer A)과 외부적 활동(Layer B)을 분리하여 두 레이어 간의 상관관계를 분석했습니다.

* **분석 결과**: 가치관 군집과 행동 군집 간의 유사도 측정 결과, ARI(0.0287)와 NMI(0.0378)가 매우 낮게 측정되었습니다.
* **인사이트**: 이는 "동일한 가치관을 가진 사람이라도 실제 행동 패턴은 전혀 다를 수 있음"을 의미합니다. 단순한 행동 로그 분석만으로는 포착할 수 없는 '심리적 동기' 레이어의 독자적인 존재 가치를 확인했습니다.

### 8.4 NB04: 에이전트 시뮬레이션 및 실제 데이터와의 GAP 분석

추출된 가치관 페르소나를 LLM 에이전트에 주입하여 실제 MovieLens 평점 데이터와 비교 시뮬레이션을 수행했습니다.

* **전반적 과소추정(Underestimation)**: 대부분의 가치관 군집에서 실제 인간보다 낮은 평점을 주는 경향이 발견되었습니다. 특히 '관계·가족 중심형(군집 3)'은 실제보다 **1.472점 낮은** 가장 보수적인 평가를 내렸습니다.
* **편향 전파의 부재**: NB02에서 발견된 사회성 편향이 드라마/로맨스 장르의 선호로 직접 이어지지는 않았습니다. 오히려 호러(GAP -1.504)와 코미디(GAP -1.261) 장르에서 극단적인 과소추정이 발생했습니다.
* **과적합(Over-acting) 문제**: 가치관 기반 시뮬레이션 분포와 실제 분포의 차이(JSD 0.4418)가 인구통계 기준선(JSD 0.3332)보다 **약 1.33배** 크게 나타났습니다. 이는 LLM이 부여된 가치관 설정을 너무 엄격하게 '연기'하느라, 실제 인간 취향의 유연함과 다양성을 상실했음을 보여줍니다.

### 결론 및 시사점

본 연구를 통해 LLM 기반 합성 페르소나가 인구통계를 넘어선 심리적 분석 도구로서 가능성이 있음을 확인했으나, 동시에 **사회성 편향의 과잉**과 가치관 과적합(Over-acting)이라는 명확한 한계를 발견했습니다. 향후 UX 시뮬레이션에 활용 시, 이러한 LLM 특유의 보수적 편향을 보정할 수 있는 프롬프트 설계가 필수적입니다.
